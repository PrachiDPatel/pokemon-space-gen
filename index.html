PokePod!!!

This started as a JavaFX desktop app I built for cs1302. You type in a date, it grabs the NASA APOD, then reads every pixel using a hand-rolled color classifier to find the dominant hue — and gives you 3 matching Pokémon. Ended up being one of my favorite projects from that class.

The web version runs the same pixel-by-pixel color algorithm — rebuilt for the browser using the Canvas API instead of JavaFX's image reader.

  • JavaFX Stage → HTML/CSS
  • HttpClient + Gson → fetch API
  • PixelReader → getImageData()
  • 3 pokémon → 1 with flavor text
  • official artwork + type badges

// Manhattan distance to find closest color
int rgbDistance(int[] rgb, int r, int g, int b) {
  return Math.abs(rgb[0] - r)
       + Math.abs(rgb[1] - g)
       + Math.abs(rgb[2] - b);
